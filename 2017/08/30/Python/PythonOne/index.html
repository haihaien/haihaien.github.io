1.0 字符串内建函数
-

count()

  • 描述:用于统计字符串里面某个字符出现的次数。可选参数为在字符串搜索的开始与结束位置

  • 语法: str.count(sub,start=0,end=len(string))

####2.0 列表

  • 列表也属于序列 序列可以进行操作包括索引,切片，加，乘，检查元素
  • 列表作为类对象，是个引用类型 因此要想保留sort,extend,append 之前的值单纯的把它赋值给另一个变量是不行的,我们必须使用deep copy numList1=numList.copy()
cmp(list1,list2)
  • 描述：比较两个列表元素
list(seg)
  • 描述用于将元组转换成列表,注意：字符串也可以被转换成列表只是会转换成一个个单独分开元素

####3.0 元组

Python元组合列表类似，不同之处在于元组的元素不能修改

  • 创建空的元组 tup1=()
  • 元组只包含一个元素的时候，需要在元素后边添加逗号 tup1=(50,)
  • tuple(seg)将列表转换成元组

函数

####不定长参数函数

  • 描述: 加了星号(*)的变量名称会存放所有未命名的变量参数

####匿名函数

描述：python 使用lambda来创建匿名函数

  • lambda只是一个表达式,函数体比def简单的多
  • lambda的主题是一个人表达式，而不是一个代码块，仅仅能在lambda表达式中封装有限的逻辑进去
  • lambda函数拥有自己的命名空间，且不能访问自有参数列表之外或全局命名空间里的参数。
  • 虽然lambda函数看起来只能写一行，却不等同于C或C++的内联函数，后者的目的是调用小函数时不占用栈内存从而增加运行效率
